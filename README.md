In-Memory Database with Query Optimization

A high-performance, lightweight in-memory database built using C++, featuring B+ Trees for indexing, optimized query execution, LRU caching, and ACID transactions.

🚀 Features
1. Fast Query Execution – Uses B+ Trees for efficient indexing and retrieval.
2. SQL-like Query Processing – Basic SQL-style queries for ease of use.
3. ACID Transactions – Ensures data consistency with commit/rollback support.
4. LRU Caching – Speeds up frequently accessed data retrieval.
5. Lightweight & Optimized – Designed for low-latency operations.

🏗️ Tech Stack
C++ 17	, SQLite , B+ Trees	, LRU Cache	, CMake	

🛠️ Installation & Setup

1. Clone the Repository
   
git clone https://github.com/pavanmaske/In-Memory-Database-with-Query-Optimization.git

cd in-memory-db

3. Build the Project

mkdir build && cd build

cmake ..

make

3. Run the Database
   
./db_main

5. Run Tests
   
./runTests
